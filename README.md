# AIHaikuGenerator  
Run sip_displayer.py to get your SIPs!  

SIP stands for Short Issa Poem. This model is trained on English translations of Issa's Japanese haiku, so the training data does not adhere to the 5-7-5 syllable format.  

`v1` and `v2` of the model use characters (see below) as their smallest building blocks. `v3` uses words (see below) as its smallest building blocks.  

The model save files for v3 are too large to upload to the repo, so the `ckpt` directory is included in [.gitignore](.gitignore).  
  
[HackRPI Devpost](https://devpost.com/software/short-issa-poem-sip-generator-lstm-wura3e) (v1 and v2)  

[Siraj Raval's](https://www.youtube.com/channel/UCWN3xxRkmTPmbKwht9FuE5A) [tutorial](https://www.youtube.com/watch?v=ZGU5kIG7b2I) on LSTMs was a huge help for this project!
